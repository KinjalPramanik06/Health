HealthGuard Pro: Advanced First Aid System

Welcome to HealthGuard Pro, a comprehensive first-aid web application designed to provide immediate, accessible instructions for standard pre-hospital scenarios. When an emergency strikes, time and accurate information are everything. This project aims to bridge the gap between panic and action by delivering a highly organized database of medical protocols directly in the browser.

🚀 Key Features

- Medical Database: Covers a massive array of critical categories including Cardiac, Respiratory, Trauma, Burns, Toxicology, and Pediatric emergencies.
- Clear, Actionable Guidance: Every emergency scenario is broken down into structured, step-by-step actions, accompanied by vital "DON'T" warnings to prevent dangerous mistakes during stressful moments.
- Rapid Retrieval System: Features an integrated chat-style interface and quick-access chips (e.g., "CPR steps", "burn treatment", "heart attack symptoms") to find life-saving information instantly.
- Modern UI/UX: Built with a sleek, responsive design featuring a dynamic light and dark theme, glassmorphism elements, and smooth CSS animations (like the SOS pulse) to maintain clarity and focus.

💻 Tech Stack

- Developed utilizing core web technologies to ensure lightweight, fast performance without the need for heavy external dependencies:
- HTML5 for semantic, accessible structure.
- CSS3 for custom variables, fluid animations, and responsive theming.
- JavaScript for dynamic data rendering, modal interactions, and fast search logic.
